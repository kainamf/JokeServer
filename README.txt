Como usar:
1. Run Python File 'server.py'
2. Acesse a porta 2000 da rede local utilizando o comando 'telnet 127.0.0.1 2000' no seu terminal
3. Escolha uma cadeira digitando um número de 1 a 10 ou digite "sair" para encerrar.
3.1 Digite qualquer tecla para começar a receber as charadas.
3.2 Digite qualquer tecla para dar continuidade na execução da(s) charada(s)
4. Digite 'sim' para mais charadas ou 'sair' para encerrar.
